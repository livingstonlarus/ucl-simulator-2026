# UCL Simulator 2026/2027

A modern web application to simulate the UEFA Champions League 2026/2027 qualifications and tournament.

## Features
- Real logos fetched from Wikipedia
- Real-time persistence with SQLite
- Responsive design for mobile and desktop
- Automatic score generation

## Tech Stack
- React (Vite)
- Express.js (Node.js)
- SQLite3
- Vanilla CSS

## Getting Started
1. Run the backend: `node server.js`
2. Run the frontend: `npm run dev`
3. Open `http://localhost:5173`
